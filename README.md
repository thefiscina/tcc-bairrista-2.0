## usar

### Pré-requisitos

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone o repositório 
   ```sh
   git clone https://github.com/thefiscina/bairrista-tcc-web.git
   ```
2. instale os pacotes > NPM packages
   ```sh
   npm install
   ```
3. Rode no navegador
   ```sh   
    ng serve -o;
   ```
